Virus

Self-replicating code that attaches to legitimate files. Spreads when infected files are opened or shared and can corrupt, delete or encrypt data.

// KEY FACTS
  • Requires user action to spread (opening a file)
  • Attaches itself to existing legitimate files
  • Can corrupt or delete data as its payload
  • Detected by anti-malware signature scanning
